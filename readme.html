» Usage Instructions
General Usage
  1. Open wp-content/themes/<YOUR THEME NAME>/footer.php
  2. Add Anywhere:
    <?php if(function_exists('wp_pagenavi')) { wp_pagenavi(); } ?>

    Default: wp_pagenavi('', '', '', '', 5, false);

    The first value: The text to appear before the navigation
    The second value: The text to appear after the navigation.
    The third value: The label for previous page.
    The fourth value: The label for next page.
    The fiveth value: Is the number of page numbers to be shown.
    The sixth value: Always show the navigation bar even it contains 1 and only 1 page.

    If you want a drop down menu style of page navigation, just replace wp_pagenavi() with wp_pagenavi_dropdown().
